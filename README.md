一个php实现单向链表结构cache的模拟程序，容量是100。
功能有：
	1.每秒age字段加1
	2.在随机位置插入新增数据
	3.删除条件：
			1).age大于10
			2).age没有大于10的数据且数据量到100，则删除第一个数据
	4.数据的id模拟ai自增长。
	5.数据存储在同文件夹下的cache_test.txt中，若存在则自动改写，若不存在则创建。

注意：
	保证用户对该文件夹的读写权限是正确的。
	$list中有原始的50条数据。
	链表最后数据的item指向null。

数据结构：
	$array={
		'id' => 0,
		'age' => 1,
		'item' => 1
	}
